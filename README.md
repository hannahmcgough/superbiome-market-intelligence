# Superbiome Market Intelligence

Weekly barrier-repair and microbiome-adjacent market intelligence across Cleanse, Treat, and Hydrate.

Strategy and analysis by [HM Studio](https://hannahmcgough.com).

## Local development

Requires Node.js 22.13 or later.

```bash
npm install
npm run dev
```

The dashboard application lives in `app/`. Its weekly data snapshot is stored in `data/market-intelligence.json`.
